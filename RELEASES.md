# Releases

This repository contains Kodi add-on sources. Download the repository ZIP from GitHub Releases and add it to Kodi.

## How to Use

1. Go to Kodi Settings → File manager → Add source
2. Enter the repository ZIP download link
3. Or use the raw GitHub URL format below

**Repository URL for Kodi:**
```
https://github.com/ashtonhardy555-stack/Should/archive/refs/heads/main.zip
```

OR (once releases are created):
```
https://github.com/ashtonhardy555-stack/Should/releases/download/v1.0.0/repository.xxx-1.0.0.zip
```
